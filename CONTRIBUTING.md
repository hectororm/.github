# Contributing to Hector ORM

Thank you for considering contributing to Hector ORM!

All Hector ORM packages are developed in a single **monorepo**. The individual package repositories
(`hectororm/collection`, `hectororm/orm`, `hectororm/query`, etc.) are **read-only mirrors** that are synchronized
automatically and do not accept issues or pull requests.

**All contributions (bug reports, feature requests, pull requests) must be made on the main repository:**

➡️ **https://github.com/hectororm/hectororm**

Please read the full contribution guide there:

➡️ **https://github.com/hectororm/hectororm/blob/main/CONTRIBUTING.md**

See also our [Code of Conduct](CODE_OF_CONDUCT.md) and [Security Policy](SECURITY.md).
